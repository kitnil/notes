* abduco
  - create session
    : abduco -c test bash

  - delete session
    : only by closing the program inside session

  - detach session
    : C-\

  - attach to session
    : abduco -a test

  - override detach session keybinding
    : abduco -e ^q -c test bash
    ^q is C-q

  - create session without attaching
    : abduco -n test htop
